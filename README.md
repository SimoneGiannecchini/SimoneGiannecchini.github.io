# Simone Giannecchini

Portfolio personale di **Simone Giannecchini**, sviluppatore web con passione per frontend, design e progetti digitali.

## Obiettivo

Questo sito raccoglie una presentazione personale, una selezione di progetti e una pagina contatti per collaborazioni o nuove opportunita.

## Struttura

- `index.html`: home page con bio e presentazione.
- `porfolio.html`: raccolta dei progetti pubblicati.
- `contact.html`: modulo contatti con integrazione EmailJS.
- `assets/scss/style.scss`: sorgente degli stili.
- `assets/scss/style.css`: CSS compilato usato dal sito.
- `assets/JS/javascript.js`: gestione del form di contatto.
